# Reasoning.py 패턴 통합 가이드

## 📋 개요

Reasoning.py 패턴은 **여러 장의 카드를 해석할 때 단순 나열이 아닌 '연결성'과 '흐름'을 읽어내는** 고급 RAG 로직입니다.

### 핵심 원칙

```
단일 카드 해석 ❌  "달 카드는 불확실성을 의미합니다."
멀티 카드 해석 ✅  "과거의 달 카드가 보여준 혼란이 → 현재 마법사 카드의 명확한 실행력으로 전환되고
                   → 미래 세계 카드의 완성으로 귀결됩니다."
```

---

## 🎯 구현된 주요 기능

### 1. Position 기반 검색 (같은 카드, 다른 위치 = 다른 해석)

#### Before (Inference.py)
```typescript
const query = `${card.nameKo} 카드가 ${card.isReversed ? '역방향으로' : '정방향으로'} 나왔을 때, ${questionDomainHint}에 대한 해석은?`;
```

#### After (Reasoning.py)
```typescript
const positionHint = card.position ? `${card.position} 위치에서` : '';
const query = `${card.nameKo} 카드가 ${positionHint} ${card.isReversed ? '역방향으로' : '정방향으로'} 나왔을 때, ${questionDomainHint}에 대한 해석은?`;
```

**효과:**
- "달" 카드 + "과거" 위치 → "지나간 혼란과 불확실성"
- "달" 카드 + "미래" 위치 → "다가올 직관적 통찰과 숨겨진 진실"

### 2. 카드 간 '케미' 분석 (`analyzeCardChemistry()`)

#### 분석 항목

**A. 메이저/마이너 비율**
```
🎯 메이저 아르카나 2장 (67%) — 큰 전환점, 인생의 중요한 국면
📅 마이너 아르카나 1장 (33%) — 일상적 사건, 세부적 흐름
```
- 메이저 50% 이상 → 인생의 중요한 전환점
- 마이너 100% → 일상적 사건, 구체적 행동 필요

**B. 원소(Element) 분포**
```
🔥💧🌬️🌿 원소(Element) 에너지 분포:
  • 물 원소 2장 (67%) — 감정·직관·흐름
  • 공기 원소 1장 (33%) — 사고·소통·분석

⚡ 지배 원소: 물 (67%) — 이 스프레드는 물 에너지가 강하게 작용합니다
```

| 원소 | 슈트 | 의미 | 지배 시 메시지 |
|------|------|------|--------------|
| 불🔥 | Wands | 열정·행동·창의 | "행동으로 옮길 때입니다" |
| 물💧 | Cups | 감정·직관·흐름 | "마음의 소리에 귀 기울이세요" |
| 공기🌬️ | Swords | 사고·소통·분석 | "명확한 판단이 필요합니다" |
| 흙🌿 | Pentacles | 물질·안정·실행 | "실질적 결과에 집중하세요" |

**C. 수비학 패턴 (같은 숫자 반복)**
```
🔢 수비학 패턴 (같은 숫자 반복):
  • 숫자 3 (여황제, 완드의 3) — 성장, 확장, 표현
```

| 숫자 | 의미 | 반복 시 메시지 |
|------|------|--------------|
| 1 | 새로운 시작, 잠재력 | "여러 영역에서 새 출발" |
| 2 | 균형, 선택 | "중요한 선택의 기로" |
| 3 | 성장, 확장 | "협력과 창의적 표현" |
| 4 | 안정, 구조 | "탄탄한 기초 구축" |
| 5 | 변화, 도전 | "성장통과 변화의 시기" |
| 7 | 성찰, 평가 | "내면 탐구 필요" |
| 10 | 완결, 순환 | "한 챕터의 완성" |

### 3. 에너지 흐름 연결 (Multi-Card Reasoning)

#### 프롬프트 구조 (2장 이상)

```
🌊 **멀티 카드 에너지 흐름 연결 (Reasoning.py 패턴):**
   3장의 카드를 단독으로 해석하지 말고, 전체 흐름을 유기적으로 연결하세요!

   📍 흐름 구조: "과거 → 현재 → 미래"

   ✨ 에너지 흐름 예시:
      "탑 (과거)"가 보여준 에너지가
      → "죽음 (현재)"의 상황으로 이어지고
      → "별 (미래)"의 결과/조언으로 귀결됩니다

   🎭 스토리텔링 원칙:
      1. 각 카드의 의미를 먼저 짧게 설명
      2. 과거→현재→미래 흐름 중심으로 종합 이야기 구성
      3. 카드들 사이에 모순이 있다면 → 극복 조언으로 풀어서 설명
      4. overallInterpretation에서 반드시 전체 흐름을 명시

   ⚠️ 중요: 각 카드의 **position(위치)** 의미를 해석에 반드시 반영하세요!
```

### 4. 한 줄 요약 조언 (Reasoning.py 고도화)

#### conclusion 필드 구조 (5파트)

```
💎 한 줄 요약 조언 (50-70자): 이 리딩의 핵심을 한 문장으로 압축한 행운의 메시지
  → "혼란 끝에 찾아올 희망을 믿고 새로운 길로 나아가세요"

🔮 핵심 메시지 (100-130자): 스프레드 전체가 말하는 가장 중요한 한 가지

✨ 실천 조언 3가지 (각 60-80자): 구체적이고 실행 가능한 행동

⚠️ 주의할 점 (60-90자): 카드가 경고하는 함정

💌 격려 메시지 (90-120자): 따뜻하고 힘이 되는 마무리
```

**한 줄 요약의 중요성:**
- 사용자가 리딩을 기억할 때 가장 먼저 떠올리는 핵심 메시지
- SNS 공유, 일기 기록 시 활용
- 복잡한 멀티 카드 리딩을 한 문장으로 압축

## 🎓 고급 전략 (추가 최적화 방향)

### 1. Position별 가중치 조정

특정 스프레드에서는 position의 중요도가 다름:

```typescript
// 미래 구현 예시
const positionWeights = {
  'Celtic Cross': { 'Outcome': 1.5, 'Present': 1.3, 'Past': 1.0 },
  'Three Card': { 'Future': 1.4, 'Present': 1.2, 'Past': 1.0 }
};
```

### 2. 카드 간 시너지 점수

특정 카드 조합은 강한 시너지:

```
예:
- 달 + 고위여사제 = "무의식/신비 원형 쌍" → 시너지 +50%
- 마법사 + 세계 = "시작/완성 원형 쌍" → 완벽한 흐름 +70%
```

### 3. 모순 카드 감지 및 극복 조언

```
예:
- 탑(파괴) + 태양(기쁨) = 모순 감지
  → AI 조언: "파괴 후에 찾아오는 진정한 기쁨을 의미합니다"
```

## 🔧 트러블슈팅

### Q1: Position이 쿼리에 반영되지 않는 것 같아요
```bash
# 로그 확인
[RAG] 달 카드 search - score: 0.872, results: 2

# 쿼리 확인 (ai.service.ts:238)
const query = `${card.nameKo} 카드가 ${positionHint} ${card.isReversed ? '역방향으로' : '정방향으로'} 나왔을 때...`
```

**해결:** positionHint가 비어있는지 확인. card.position이 제대로 전달되는지 체크

### Q2: 카드 케미 분석이 표시되지 않아요
```bash
# 프롬프트 로그 확인
console.log(chemistryAnalysis);

# null이면 카드 데이터 체크
cards.forEach(c => console.log(c.suit, c.number));
```

**해결:** suit/number 필드가 null인지 확인

### Q3: 멀티 카드인데 흐름 연결이 약해요
```bash
# 카드 개수 확인
console.log(`Card count: ${ragContexts.length}`);
```

**해결:** 2장 미만이면 멀티 카드 로직 미작동. 스프레드 설정 확인
